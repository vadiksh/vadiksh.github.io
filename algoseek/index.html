  • Trades Only
    (tick)

    All Trades with millisecond or nanosecond timestamp and exchange flags.

  • Trade + Quotes/
    Level 2 (tick) with
    condition codes

    Trade+Quote is all trades and top-of-book quotes (Bid and Ask) with all flags from all exchanges with millisecond or nanosecond timestamps. Data may include National Best Bid Offer (if applicable).

  • Full Depth/
    Multiple Depth

    Full depth (or multiple depth) shows price and size for limit orders placed for Bid and Ask. Each time a limit order changes then this change is published.

  • Trade+Quote minute
    bar with 55+
    data points

    Trade+Quote minute bars provide in-depth analysis of each minute including Open/High/Low/Close for Bid/Ask/Trades, spread data, trade count at bid/mid/ask and other quantitative fields. Designed for quant clients needing in-depth intraday data for their Machine Learning.

  • Trade minute
    bars

    Trade only minute bars provide Open/High/Low/Close price, volume and number of trades based on Trades during the minute bar period.

  • Daily OHLC

    Daily Open/High/Low/Close is the daily price and size for official open/close with the high/low of the day.

  • Custom bars

    Algoseek’s parallel infrastructure framework enables the quick development and processing of big data sets to create custom bars and datasets to meet our client’s needs.

  • EOD

    End of Day is a dataset based on data from the complete trading day.

  • Reference
    Data

    Reference data covers a wide range of meta data for a dataset, for example with equity it can inlcude corporate actions, type of equity, listing dates, primary exchange, etc. Algoseek maintains multiple Reference Datasets, for example Security Master for equity.
