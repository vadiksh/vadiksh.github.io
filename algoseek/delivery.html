Сloud

Algoseek keeps most of its data in S3 cloud, so anyone using AWS can easily browse and download our cloud-based data using available tools. We provide the cross-account data access, so you can utilize it in your existing AWS infrastructure with almost zero overhead.

Note: this option is only available for Institutional Subscription clients.

AWS CLI

Using AWS Command Line Interface (AWS CLI), you can download your data directly from AWS S3 (Simple Storage Service) , a popular and reliable object storage solution with a wide range of applications.

This option is great for medium-sized (above 50 Gb) and large orders (> 250 Gb).

AWS SDK

FTP

We can provide data for download from our dedicated FTP server.
You can use 'ftp' command line tool to easily browse and download individual files. Alternatively, you can download FTP-provided data from your web browser.

Please, contact our support team to get more details about this delivery method.

API

Algoseek provides API delivery in its majority-owned subsidiary QuantGo.

QuantGo Python API is a Python library that provides easy access to QuantGo data in Amazon cloud and is built on top of Boto3 library. API is compatible with Python 2.7.X and 3.2 or higher.
