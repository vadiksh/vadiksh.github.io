Purchase Options

AlgoSeek provides a range of purchase options suitable for independent traders to large banks.

Purchase Options
How it works
Pros
Cons
  • Buy

    Pay full price and own the data in perpetuity.

    For Clients who want to own the data, this option is less expensive than leasing and then buying.

    Higher upfront fees than Lease or Rent.

  • Lease

    Lease the historical data and updates with an option to buy when the lease expires. Leases are subject to qualification.

    Lowers the upfront and monthly update fees.

    Algoseek Data must be deleted at the end of the Lease. Overall cost of owning data is higher if client exercises option to purchase data at end of Lease instead of buying data

  • Rent

    Access in the AWS Cloud with QuantGo.com's Virtual Quant Lab. Launch remote Amazon AWS computer instances and install any software, application or codes and download research results.

    Most affordable access option.

    Can only access the data on AWS Computer instances in a controlled environment. Data cannot be used on Client's own infrastructure or computers. Client also pays for all AWS Cloud computer costs.
