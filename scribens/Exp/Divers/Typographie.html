Règles typographiques

 

Règles typographiques :

- Pour le point : texte.{espace}texte

Ex : Je ne m'attendais pas à ce que vous veniez si tôt. Je n'ai pas fini de travailler.

- Pour la virgule : texte,{espace}texte

Ex : Tu me fatigues, je rentre chez moi.

- Pour le point-virgule : texte{espace};{espace}texte

Ex : Elle me manque ; j'espère que je lui manque aussi.

- Pour les deux-points : texte{espace}:{espace}texte

Ex : Nous devons respecter la règle suivante : on ne se tape pas dessus avant midi.

- Pour le point d'interrogation : texte{espace}?{espace}texte

Ex : Va-t-il mieux aujourd'hui ? Je me suis beaucoup inquiété.

- Pour le point d'exclamation : texte{espace}!{espace}texte

Ex : Nous y parviendrons ! J'en suis à peu près sûr.

- Pour les points de suspension : texte...{espace}texte

Ex : Cette approche me parait délicate... Nous devons agir avec prudence.

- Pour les guillemets : texte{espace}«{espace}texte{espace}»{espace}texte

Ex : On l'appelait « l'idiot du village » mais il nous dédaignait totalement.

- Pour les parenthèses : texte{espace}(texte){espace}texte

Ex : Ce qui le distinguait de ses frères (d'après ses parents), c'était son extrême douceur.

- Pour les crochets : texte{espace}[texte]{espace}texte

Ex : Elle [Marie] fréquentait un charpentier de Nazareth.

- Pour les tirets : texte{espace}–{espace}texte

Ex : Cette mine a beaucoup de ressources – fer, cuivre et aluminium – mais elle est peu exploitée.
