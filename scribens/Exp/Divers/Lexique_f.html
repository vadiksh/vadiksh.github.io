Lexique

 

1. Le sujet

- Le sujet est le mot ou l’ensemble de mots dont on parle et avec lequel le verbe s’accorde. Le sujet précède souvent le verbe.

Le sujet est très souvent représenté par un nom commun, un nom propre ou un pronom.

Ex : Il mange une banane.

Ex : Le jardinier coupe les fleurs.

Ex : Samuel arrive à marcher.

- Il peut aussi être représenté par un infinitif.

Ex : Faire du sport est excellent pour la santé.

Ex : Se balancer sur cette branche est agréable.

- Il peut prendre une tournure impersonnelle avec l’emploi de il.

Ex : Il pleut beaucoup.

Ex : Il y a des changements.

2. Le COD (complément d’objet direct)

Le complément d’objet direct représente l’objet de l’action du verbe.

Pour le reconnaître, on pose après le verbe la question : qui ? quoi ?

Ex : J’ai mangé une pomme. (mangé quoi ?)

Ex : Il regarde ce film. (regarder quoi ?)

Ex : Il la raccompagne. (raccompagner qui ?)

Il peut être désigné par :

- Un nom. Ex : Il change sa belle chemise.

- Un pronom. Ex : Il le modifie.

- Un verbe à l’infinitif. Ex : Il veut se baigner. (vouloir quoi ?)

- Une proposition subordonnée. Ex : Il dit qu’il est intéressant. (dire quoi ?)

3. Le COI (complément d’objet indirect)

Le complément d’objet indirect représente l’objet de l’action et est précédé d’une préposition.

Pour le reconnaître, on pose après le verbe la question : de qui ? de quoi ? à qui ? dans quoi ? etc.

Ex : Il change de vélo. (de quoi ?)

Ex : Il donne à Françoise un bonbon. (à qui ?)

Ex : Elle se moque d’eux. (de qui ?)

Il peut être désigné par :

- Un nom. Ex : Il marche dans la rue.

- Un pronom. Ex : Il lui donne ce cadeau. (= donner à lui)

- Un verbe à l’infinitif. Ex : Elle choisit de se lancer un nouveau défi. (de quoi ?)

- Une proposition subordonnée. Ex : Ils obéissent à ce qu’il demande.

Le complément d’objet indirect peut être placé avant le verbe :

Ex : Il lui demande de continuer.

Ex : Le bâton avec lequel il marche est très solide.

4. L’épithète

L’épithète est un mot apportant une précision sur le nom : qualité, opinion, état. Il s’accorde en genre et en nombre avec lui.

- Il peut être un adjectif qualificatif :

Ex : Un beau chapeau.

Ex : C’est un garçon gentil et drôle.

- Ou un participe passé :

Ex : Il aime les affiches collées par ses amis.

Ex : Elle déteste ses propos détournés.

5. L’adverbe

Un adverbe est un mot invariable dont le rôle est d’apporter une nuance à un verbe ou un adjectif.

Ex : Il court rapidement. (modifie le verbe)

Ex : Ma cousine lit très bien. (modifie le verbe)

Ex : Une très grande voile. (modifie l’adjectif)

Ex : Il assiste à cette si belle fête. (modifie l’adjectif)

Les adverbes peuvent être de plusieurs types :

- Adverbes de manière : Elle s’assied tranquillement.

- Adverbes de lieu : Je vient ici.

- Adverbes de temps : Il arrive bientôt.

- Adverbes de quantité : Peu d’huile coule.

Une locution adverbiale est une expression ayant le rôle d’adverbe.

Il chante tout le temps. (peut être remplacé par toujours)

Etienne travaille d’arrache pied pour retrouver le bandit. (peut être remplacé par beaucoup)

6. Le déterminant

Le déterminant fait partie du groupe nominal. Il s’accorde en genre et en nombre avec celui-ci et est toujours placé avant le nom.

Ex : Ce chien.

Ex : Un grand train.

Ex : Des cerisiers.

Les déterminants peuvent être de plusieurs types :

- Articles : la, un, une, du, des, etc.

Ex : Une voiture.

Ex : La vitrine.

- Déterminants possessifs : sa, son, ses, mon, mes, etc.

Ex : Ses bijoux.

Ex : Leurs bijoux.

- Déterminants démonstratifs : ce, ces, cette, etc.

Ex : Cette main.

Ex : Ce bracelet.

- Déterminants indéfinis : quelques, plusieurs, certains, etc.

Ex : Certains bâtiments

Ex : Quelques jolies perles.

- Déterminants interrogatifs et exclamatifs : Quel, quelles, etc.

Ex : Quel temps fera-t-il demain ?

Ex : Quelle belle journée !

- Déterminants numéraux cardinaux : neuf, quatre-vingts quinze, etc.

Ex : Il a 25 voitures.

Ex : 2 soldats approchent.

7. Les conjonctions

- Une conjonction de subordination est un mot invariable ayant pour rôle de lier deux propositions.

Ex : Il prends le train, car il doit rejoindre sa sœur.

Ex : Le jardinier sortira si le temps le permettra.

- Une conjonction de coordination est un mot invariable reliant deux groupes de mots.

Ex : Cet avion est gris et imposant. (relie deux adjectifs)

Ex : Régis et Sophie seront à cette soirée. (relie deux noms propres)

Ex : Il court ou écoute de la musique en même temps. (relie deux verbes)

8. Les prépositions

Une préposition est un mot invariable permettant de relier deux mots.

Ex : Le vélo de Sandrine.

Ex : Il arrive à Munich.

Ex : Dès le lendemain, il sera ici.

Les prépositions peuvent être de plusieurs types :

- Prépositions de lieu : dans, chez, sous, sur, etc.

Ex : Prends la clé sous le paillasson.

Ex : Il va chez le médecin.

- Prépositions de temps : après, avant, dès, etc.

Il viendra après la fête.

Avant le discours, il y croyait encore.

- Prépositions d’état : en, avec

Un parquet en sapin.

- Prépositions de cause : pour, vu, etc.

Il quitte le bâtiment, vu la foule immense.

- Prépositions de but : pour, envers, etc.

Il milite pour le droit des animaux.

9. Les pronoms

Les pronoms ont pour rôle de remplacer le nom.

Ex : Il monte dans le train. (remplace le nom d’une personne)

Ex : Celui-là est intéressant. (remplace le nom d’un objet)

Les pronoms peuvent être de plusieurs types :

- Pronoms personnels : je, tu, lui, etc.

Ex : Je change de vélo.

Ex : Fabrice danse avec elle.

- Pronoms possessifs : le sien, le mien, etc.

Gérard veut le mien.

Le vôtre sent bon.

- Pronoms interrogatifs : lequel, lesquelles, qui, etc.

Qui fait la cuisine ?

Lequel est le meilleur ?

- Pronoms démonstratifs : celui-ci, celui, celle, etc.

Celui qui lance très haut est un bon athlète.

J’aime celle-ci.

- Pronoms relatifs (introduit une subordonnée) : qui, dont, à laquelle.

La table qui est bleue est ma préférée.

Le courage dont il fait preuve m’impressionne.

10. Les interjections

Une interjection est un mot invariable servant souvent à exprimer une émotion, un sentiment ou une réaction. Elle est souvent accompagnée d’un point d’interrogation.

Ex : Silence !

Ex : Aïe !

Ex : Ouf !
